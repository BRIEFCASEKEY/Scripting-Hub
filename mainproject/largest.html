Lesson 9: C program to find Largest of Three Numbers

In C program, to find the largest of three numbers, use if-else statements to compare them and identify the greatest.
Example in C:
#include<stdio.h>
int main() {
  int a=12, b=25, c=7;
  if(a>b&&a>c)
    printf("Largest = %d", a);
  else if(b > c)
    printf("Largest = %d", b);
  else
    printf("Largest = %d", c);
  return 0;
}
