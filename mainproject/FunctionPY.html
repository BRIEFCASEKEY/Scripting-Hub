Lesson 6: Functions

In this lesson, we learn about functions which are super helpful in Python. Functions are blocks of code that we can use at anytime by doing something called "calling" it. If you want to do a task over and over in separate parts of program, you would make a function to perform the task, so instead of writing the code again, you can simply call the function. For example:

def example():
    print("This is a function")

To make a function, we type def, put a space, then the name of our function. After the name you put an opening and closing bracket, then a colon. Where I printed something you can write code, as long as it is indented (4 spaces). In order to run the function, we have to "call" it. To call the function we made above, we would type the following:
example()
As you can see, we type the name of the function and put the brackets. Now the function should look like this:

def myfunction():
    print("hello world")
myfunction()
