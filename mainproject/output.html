Lesson 1: Output

A great place to start in C++ is what is often referred to as "Printing" in other languages. In C++, this is done with "cout". C++ is quite different from other languages in some aspects. We are required to have a "Main Function" in C++ code, which is the start of the program. Here is an example that would put "Hi" into the output:

#include<iostream>
int main() {
std::cout<<"hi";
}

Just like in C, we use iostream instead of stdio.h. It is the header file in C++. You can see we use "std::" before cout. Normally in C, we would just use print but in C++ it's different. A namespace is compulsory for the code to run so "std" is a namespace in C++ which is used to access standard library tools so input/output are stored in it.
