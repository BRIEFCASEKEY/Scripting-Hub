Lesson 3: Math

Now we will learn to do math in Python. You can type stuff like in normal math. Some examples
1+1
1-1
1*1
1/1
The first and second one we add and subtract 1 and 1 by using plus and minus sign. The third one we multiply both numbers by "*" sign. The fourth one we divide both numbers by "/" forward slash. If you do not want decimals and just the quotient you can use double forward slashes "//". If you only want the remainder you can use "%" percentage sign.
1//1
1%1
This does not work for multiplication. We can print the following sum:
print(1+1)
We do not use ("") because those double quotation marks will indicate to print the "1+1" and do not do the maths. Now from the above example find the multiplication of 4 and 5.
