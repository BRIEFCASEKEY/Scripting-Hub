Lesson 13: C program to find sum of elements of an array

In C programming, an array is a collection of elements of the same data type stored in contiguous memory locations.

For example:
#include<stdio.h>
int main() {
int arr[5] = {10, 20, 30, 40, 50};
int sum = 0;
for (int i = 0; i < 5; i++)
sum += arr[i];
printf("Sum = %d", sum);
return 0;
}
