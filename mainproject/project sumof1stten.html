Lesson 11: C program to Find Sum of First 10 Natural Numbers

Example in C:
#include<stdio.h>
int main() {
  int sum = 0;
  for(int i = 1; i <= 10; i++)
    sum += i;
  printf("Sum = %d", sum);
  return 0;
}
