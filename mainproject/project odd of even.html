Lesson 7: C program to Check Even or Odd

In C Program, it checks if a number is odd or even by using num % 2—if the remainder is 0, it’s even; otherwise, it’s odd.
Example in C using a fixed number:
#include<stdio.h>
int main() {
  int num = 27;
  if(num % 2 == 0)
    printf("Even Number");
  else
    printf("Odd Number");
  return 0;
}
