Lesson 12: C program to Print Fibonacci Series up to n terms

In C program, to print the Fibonacci series up to n terms, start with 0 and 1 and use a loop to add the last two numbers repeatedly until n terms are printed.
Example in C (n = 10):
#include<stdio.h>
int main() {
  int n = 10;
  int a = 0, b = 1, c;
  printf("%d %d ", a, b);
  for(int i = 3; i <= n; i++) {
    c = a + b;
    printf("%d ", c);
    a = b;
    b = c;
  }
  return 0;
}
