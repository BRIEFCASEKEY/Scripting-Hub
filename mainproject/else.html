Lesson 10: Else

Now we talk about else, which is used if an if statement is not met. This is very useful for actual logic in your code, making certain events happen if conditions are met, and if not met, make a different event happen. For example:

#include<iostream>
int main() {
if (false) {
std::cout<<"Condition met";
} else {
std::cout<<"Condition not met";
}

This is a normal if statement, however we have added else to the end curly bracket of the initial if statement. Then, we have another set of curly brackets, so if it does end up going to else, everything in those curly brackets will run.
