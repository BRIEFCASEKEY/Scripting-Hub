Lesson 5: Const

Now we can learn about using the const coding term. Const is yet another type of variable we can use in JavaScript, meaning constant. A constant value can never be changed. For example, if we use var/let, we could change the variable later to be something else. When you use const it is a constant variable, so the value can not be changed. For example

const word="hello"
console.log(word)

Now from above code display hello world.
