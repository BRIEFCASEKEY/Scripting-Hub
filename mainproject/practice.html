C Basics

Variable
A named memory location to store data. Example: int x = 5;
Data Type
Defines the type of data a variable can hold. Example: int, char, float.
Loop
Used to repeat a block of code multiple times. Example: for, while.
Function
A reusable block of code that performs a task. Example: int sum(int a,int b){return a+b;}

C++ Basics

Class
Class is a blueprint for creating objects. Example: class Car {...}
Objects
An object is an instance of a class. It represents a real-world entity created from a class blueprint. Example: Car myCar; ...
Constructors
A constructor is a special function in a class that is automatically called when an object is created Example: class Car{...} Car myCar("");
Functions/Methods
Functions (or methods inside classes) are blocks of code that perform specific tasks. Example: int add(int a, int b) {return a+b;}

JavaScript Basics

Variable
Stores data. Example: let x = 10;
Function
Reusable block of code. Example: function add(a,b){return a+b;}
DOM
Document Object Model for interacting with HTML elements.
Event
Action detected by JS. Example: onclick, onchange.

Python Basics

Variable
Stores data. Example: x = 5
List
Ordered collection of items. Example: nums = [1,2,3]
Dictionary
Key-value pairs. Example: person = {"name":"John"}
Function
Reusable block of code. Example: def add(a,b): return a+b
