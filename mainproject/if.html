Lesson 9: If Statement

Next we will cover if statements, a very important part of C++. If statements will run certain code if a condition is met. For example, you could check if 2+2 equals 4, and if it does, run some code. For example:

#include<iostream>
int main() {
if (true) {
std::cout<< "condition met";
}}
An if statement has 3 main parts on its first line: if, condition, and the opening curly bracket. To start any if statement, you write "if". You then put your condition in parentheses, such as (2>1). Then you put your opening curly bracket, which has an end curly bracket anywhere you want the if statement code to end. Everything inside the curly brackets will run if the condition is met.
