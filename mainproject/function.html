Lesson 7: Functions

Functions are super helpful.They are basically block of codes that have a certain purpose. If you have a line of code you keep using over and over, you can make a function for it and simply just use the function every time instead of writing out the whole line again. First, to use a function we need to write a function:

function myfunction() {
    console.log("hello")
}

Now every time we use this function, it will log "hello" in the console. But, the function will not run just like that. We need to invoke or "call" the function. When you call a function, it will run the code in the function then continue the script where it left off. To call our example function, we will type the following right below it:
myfunction()
So now the code should look like this: (4 spaces are indents!)

function myfunction() {
    console.log("hello")
}
myfunction()
