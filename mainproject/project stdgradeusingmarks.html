Lesson 10: C program to Find Student Grade Using Marks

in C program,to find a student’s grade, compare their marks against set ranges using if-else statements and assign the corresponding grade.
Example in C:
#include<stdio.h>
int main() {
  int marks = 78;
  if(marks >= 90)
    printf("Grade A");
  else if(marks >= 75)
    printf("Grade B");
  else if(marks >= 60)
    printf("Grade C");
  else if(marks >= 40)
    printf("Grade D");
  else
    printf("Grade F");
  return 0;
}
