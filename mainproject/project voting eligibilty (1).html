Lesson 8: C program to Check Voting Eligibility

In C Program, to check voting eligibility, use an if statement to see if the person’s age is 18 or above—if yes, they’re eligible; otherwise, they’re not. Example in C:
#include<stdio.h>
int main() {
  int age = 17;
  if(age >= 18)
    printf("Eligible to Vote");
  else
    printf("Not Eligible");
  return 0;
}
