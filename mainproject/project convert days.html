Lesson 5: Convert Days into Years, Weeks, and Days

In C program, to convert days into years, weeks, and days in C, we divide the total days by 365 to get years, then use the remainder to find weeks using division by 7, and the final remainder gives leftover days. Formula to convert days:
Years = days / 365
Weeks = (days % 365) / 7
Days = (days % 365) % 7

Example:
#include<stdio.h>
int main() {
  int days = 500;
  int years = days / 365;
  int weeks = (days % 365) / 7;
  int remDays = (days % 365) % 7;
  printf("Years = %d\\nWeeks = %d\\nDays = %d", years, weeks, remDays);
  return 0;
}
